# 亚马逊运营新手学习文档

> 适用对象：已申请或刚通过审核的亚马逊卖家，运营经验 0-3 个月  
> 目标：在 4-8 周内独立完成「选品 → 上架 → 发货 → 出单 → 基础优化」全流程

---

## 一、开店前你需要知道的事

### 1.1 亚马逊卖家模式

| 模式 | 说明 | 新手建议 |
|------|------|----------|
| **FBA**（Fulfillment by Amazon） | 货物入亚马逊仓，由平台仓储、配送、处理退换货 | 优先选择，可获得 Prime 标识，转化更高 |
| **FBM**（Fulfillment by Merchant） | 卖家自行发货 | 适合大件、定制、测款或利润空间极大的品类 |
| **专业卖家计划** | 月租约 $39.99/月 | 月销 40 件以上更划算，功能更全 |
| **个人卖家计划** | 无月租，每件约 $0.99 | 仅适合极低销量测试 |

### 1.2 主要站点选择

| 站点 | 特点 | 适合 |
|------|------|------|
| **美国站** | 市场最大、竞争最激烈 | 大多数新手首选 |
| **欧洲站**（英/德/法/意/西等） | 多国统一账户，需注意 VAT | 有欧洲合规能力时 |
| **日本站** | 客单价高、规则严格 | 有日语或本地化能力时 |
| **加拿大/墨西哥** | 与北美账户联动 | 北美扩张时 |

**新手建议**：先专注一个站点（通常美国站），跑通流程后再扩展。

### 1.3 必备材料清单

- [ ] 营业执照（企业或个体工商户）
- [ ] 法人身份证正反面
- [ ] 双币信用卡（Visa/Master，有效期 6 个月以上）
- [ ] 收款账户（如 PingPong、万里汇、Payoneer 等）
- [ ] 企业邮箱 + 手机号
- [ ] 部分站点可能要求：银行对账单、地址证明、产品资质（如 FDA、CE 等）

### 1.4 主要费用（心里有数）

| 费用类型 | 大致范围 | 说明 |
|----------|----------|------|
| 月租费 | ~$39.99/月 | 专业卖家计划 |
| 销售佣金 | 8%-15% | 因品类而异 |
| FBA 配送费 | 按尺寸重量计 | 后台有计算器 |
| FBA 仓储费 | 按月/立方英尺 | 旺季更高，注意滞销 |
| 广告费 | 自定预算 | 按点击付费（CPC） |
| 头程物流 | 自定 | 中国发至亚马逊仓 |

## 二、Seller Central 后台核心模块

登录地址：各站点 Seller Central（如 sellercentral.amazon.com）

### 2.1 必会模块速查

| 模块 | 路径/名称 | 用途 |
|------|-----------|------|
| 仪表盘 | Dashboard | 今日销售、账户健康概览 |
| 库存 | Inventory → Manage All Inventory | 管理 SKU、补货 |
| 货件 | Inventory → Manage FBA Shipments / Send to Amazon | 创建 FBA 入库 |
| 订单 | Orders | 查看、处理订单 |
| 广告 | Advertising → Campaign Manager | 创建和管理广告 |
| 报告 | Reports | 业务、库存、广告数据 |
| 绩效 | Performance → Account Health | **账户健康，务必每日查看** |
| 买家消息 | Customer Messages | 24 小时内回复 |
| 品牌注册 | Brand Registry | 有商标后可申请 |

### 2.2 账户健康三大核心指标

| 指标 | 英文 | 要求（参考） | 超标后果 |
|------|------|--------------|----------|
| 订单缺陷率 | ODR | < 1% | 账户受限 |
| 迟发率 | Late Shipment Rate | < 4% | 影响 Buy Box |
| 取消率 | Cancellation Rate | < 2.5% | 账户警告 |

**新手习惯**：每天打开 Account Health，有黄色/红色预警立即处理。

## 三、选品入门（新手最容易踩坑的环节）

### 3.1 新手选品原则

1. **从小做起**：首款产品建议投入可控（总成本 1-3 万人民币级别测款）
2. **避开红海大类**：如手机壳、数据线等极度内卷品类
3. **避开强合规品类**：食品、药品、儿童用品、电池等需资质
4. **有利润空间**：售价 - 采购 - 头程 - FBA - 佣金 - 广告 ≥ 25% 毛利较健康
5. **可差异化**：功能、外观、包装、组合装、配件等

### 3.2 简单选品方法

**方法一：类目浏览法**
- 在 Amazon 前台浏览目标类目 Best Sellers / New Releases
- 看评论数 100-500、评分 4.0+ 的产品（说明有需求但竞争未饱和）
- 记录价格区间、卖点、差评痛点

**方法二：关键词法**
- 用 Helium 10、Jungle Scout、卖家精灵等工具（有免费试用）
- 搜索量适中（月搜索 3000-30000）、竞争度中等的关键词
- 看首页产品评论分布是否过于集中

**方法三：供应链法**
- 从自己会做的、有货源优势的品类出发
- 1688 / 产业带工厂 → 评估能否做出差异化

### 3.3 选品红线（不要碰）

- 侵犯知识产权（仿牌、未授权图案、专利产品）
- 亚马逊禁售或限制类目
- 需要认证但无法提供的（如部分电子产品、化妆品）
- 季节性过强、保质期短
- 体积大重量重导致 FBA 费吃掉利润

## 四、Listing 创建与优化

### 4.1 创建 Listing 的方式

| 方式 | 说明 |
|------|------|
| 跟卖 | 挂在已有 ASIN 下卖同款（需有合法销售权，新手慎用） |
| 自建 Listing | 创建新 ASIN，推荐新手使用 |

路径：**Catalog → Add Products**

### 4.2 Listing 核心要素

| 要素 | 要求 | 技巧 |
|------|------|------|
| **标题 Title** | 200 字符内（因类目而异） | 核心关键词 + 属性（尺寸/颜色/数量）+ 卖点，避免堆砌 |
| **五点描述 Bullet Points** | 5 条 | 每条解决一个痛点，写好处不写参数堆砌 |
| **主图 Main Image** | 白底 RGB 255,255,255 | 产品占画面 85%+，无文字/logo |
| **副图** | 最多 7 张 | 场景图、尺寸对比、功能拆解、包装内容 |
| **A+ 页面** | 品牌备案后可用 | 图文增强转化，强烈建议做 |
| **后台关键词 Search Terms** | 250 字节 | 不重复标题已有词，用逗号分隔，不加标点 |
| **类目节点 Browse Node** | 必须准确 | 影响搜索和推荐 |

### 4.3 关键词基础

1. 用 Amazon 搜索框下拉联想词
2. 用工具反查竞品关键词
3. 标题放最高搜索量核心词，五点放长尾词
4. 不要堆砌品牌名（非自己品牌）

### 4.4 定价初探

```
建议售价 ≈ (产品成本 + 头程 + 包装) ÷ (1 - 佣金% - FBA费占比 - 目标毛利率)
```

- 上架前用后台 **Fee Preview** 估算 FBA 费用
- 新品可略低于竞品 5%-10% 获取初始订单
- 有评论后可逐步调价

## 五、FBA 发货流程（Send to Amazon）

自 2022 年起，全球统一使用 **Send to Amazon（STA）** 流程。

### 5.1 发货步骤

```
1. Inventory → Send to Amazon
2. 选择要发货的 SKU 和数量
3. 确认包装类型（混装 or 原厂包装）
4. 填写每个 SKU 的包装尺寸和重量
5. 生成并打印 FNSKU 标签（贴每个商品上）
6. 创建货件 → 选择亚马逊配送中心
7. 打印外箱标签（Box Label）
8. 联系头程物流商发货
9. 在后台填写追踪号
10. 货物签收后 1-3 天变为可售状态
```

### 5.2 贴标要点

- **FNSKU 标签**：每个商品上，覆盖原有条形码
- **外箱标签**：每箱侧面贴 2 个，避免遮挡
- **请勿覆盖** 重要信息，标签需清晰可扫描

### 5.3 头程物流选择

| 方式 | 时效 | 成本 | 适合 |
|------|------|------|------|
| 空运 | 7-15 天 | 高 | 小件、急补货 |
| 海运快船 | 20-30 天 | 中 | 常规补货 |
| 海运慢船 | 35-45 天 | 低 | 不急的大货 |
| 亚马逊跨境物流 AGL | 因线路而异 | 官方渠道 | 想简化流程时 |

### 5.4 入库常见问题

| 问题 | 预防 |
|------|------|
| 货件被拒收 | 标签错误、箱数不符、SKU 不匹配 |
| 入库延迟 | 旺季仓容紧张，提前发货 |
| 数量差异 | 发货前仔细点数，保留装箱记录 |
| 长期仓储费 | 避免滞销库存积压超过 6 个月 |

## 六、订单与客服

### 6.1 订单处理

- **FBA 订单**：亚马逊自动处理，你只需关注库存
- **FBM 订单**：需在承诺时间内发货并上传追踪号

### 6.2 客服规范

- 24 小时内回复买家消息（影响 ODR）
- 语气专业友好，先解决问题再解释
- 遇到差评：先分析原因，必要时通过「Request a Review」合规邀评
- **禁止**：引导站外交易、索要好评、威胁买家改评

### 6.3 退换货

- FBA 退换货由亚马逊处理，费用按规则扣除
- 了解各品类的退货率，计入成本模型

## 七、基础广告入门（Sponsored Products）

### 7.1 为什么新手要做广告

- 新品没有自然排名，广告是获取初始流量和销量的主要手段
- 广告数据可反哺关键词优化

### 7.2 第一款产品的广告结构（极简版）

```
Campaign 1：自动广告（Auto）
  - 预算：$10-20/天
  - 目的：跑词，发现有效搜索词
  - 运行 2-4 周后下载 Search Term Report

Campaign 2：手动广告（Manual - Exact）
  - 预算：$10-15/天
  - 投放自动广告中出单的词
  - 否定无效词（Negative Keywords）
```

### 7.3 核心指标

| 指标 | 公式 | 新手参考 |
|------|------|----------|
| ACOS | 广告花费 ÷ 广告销售额 | 初期 30%-50% 可接受（测款期） |
| CTR | 点击 ÷ 曝光 | > 0.3% 较好 |
| CVR | 订单 ÷ 点击 | > 10% 说明 Listing 不错 |
| CPC | 每次点击成本 | 因品类而异 |

### 7.4 广告禁忌

- Listing 未优化好就大量烧钱
- 不否词，预算被无效点击消耗
- 只看 ACOS 不看整体利润

## 八、合规与风险防控

### 8.1 知识产权

- 上架前做专利/商标检索（USPTO、Google Patents）
- 不要碰大牌同款、影视 IP、球队 Logo
- 收到侵权投诉（IP Complaint）立即下架并申诉

### 8.2 产品合规

- 电子产品：FCC（美）、CE（欧）
- 儿童产品：CPSIA、CPC 测试
- 化妆品：FDA 相关标签要求
- 食品：标签、成分、进口要求

### 8.3 账户安全

- 固定 IP / 专用电脑操作，避免多账户关联
- 不要刷单、刷评（Vine 除外）
- 收款账户、信用卡信息保持一致
- 定期备份重要数据

## 十一、常见问题 FAQ

**Q：多久能出第一单？**  
A：FBA 入库后 1-4 周较常见，取决于品类竞争和广告投入。不要指望上架即爆单。

**Q：没有商标能做品牌吗？**  
A：可以卖货，但无法做 A+、品牌旗舰店、Vine 等品牌权益。建议尽早布局商标。

**Q：被跟卖怎么办？**  
A：品牌备案 + Project Zero / 透明计划；或差异化、绑定配件、优化服务。

**Q：账户审核要多久？**  
A：通常 1-5 个工作日，需补材料可能更长。

**Q：新手最容易犯的错？**  
A：选品冲动、不做利润计算、忽视合规、不学习就投大量广告。
