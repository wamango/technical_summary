# 亚马逊运营学习资料

本目录为亚马逊（Amazon）卖家运营学习资料，适合已申请或即将申请店铺的中国卖家使用。

## 文档目录

| 文档 | 适合人群 | 内容概要 |
|------|----------|----------|
| [01-新手学习文档.md](./01-新手学习文档.md) | 刚开店、0-3 个月 | 开店准备、后台操作、选品、Listing、FBA、基础广告、合规 |
| [02-进阶教程.md](./02-进阶教程.md) | 有稳定出单、3 个月以上 | PPC 体系、品牌运营、数据分析、供应链、多站点、团队化 |

## 建议学习路径

```
第 1 周：通读新手文档 → 熟悉 Seller Central 后台
第 2-4 周：完成首款产品上架 + FBA 发货
第 2-3 月：掌握订单/库存/客服 + 基础广告
第 3 月后：进入进阶教程，系统化提升利润与规模
```

## 官方资源（建议收藏）

- 亚马逊全球开店（中国）：https://gs.amazon.cn
- Seller Central 帮助中心：各站点后台右上角「帮助」
- 卖家大学（Seller University）：后台内免费课程
- 亚马逊广告官方学习：https://advertising.amazon.com

## 使用说明

- 文档基于 2025-2026 年平台规则整理，政策可能更新，以官方为准
- 金额以美元（USD）为主，欧洲站注意 VAT 等税务要求
- 实操时建议边学边做，用第一款产品走完完整流程
